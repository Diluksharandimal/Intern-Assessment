🛒 Vector Vibe - Point of Sale (POS) System
A modern, full-stack Point of Sale (POS) dashboard application built for the Full-Stack Developer Intern Assignment. It features a React frontend with dynamic category filtering, live cart calculations, and a robust Node.js/Express/MongoDB backend for inventory and transaction management.

🚀 Features
Dynamic Product Catalog: Fetches live inventory data directly from MongoDB, categorized into All Products, Shoes, Clothing, and Others.

Smooth UI Interactions: Responsive dashboard layout featuring Framer Motion cascade animations and tactile UI controls.

Real-time Shopping Cart: Instantly calculates Sub-total, 12% Tax, 10% Promo Discounts, and Total Payment.

Backend Persistence: Logs transaction details and stores orders on a local Node.js server.

🛠️ Tech Stack
Frontend (Client)
React.js (Vite)

Tailwind CSS

Lucide React (Icons)

Framer Motion (Animations)

Backend (Server)
Node.js

Express.js

MongoDB & Mongoose
